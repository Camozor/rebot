<h1 align=center><code>Rebot</code></h1>

Mon petit bot Discord Rematch

## Développement

* Installer [Nix](https://nixos.org/download/) en multi user et autoriser les [flakes](https://nixos.wiki/wiki/flakes).

* Configurer les variables d'environnement nécessaires
```bash
cp .env.example .env
```

* Démarrer le projet en local
```bash
just run
```
